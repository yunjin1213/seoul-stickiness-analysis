## 샘플 데이터

원본 데이터는 용량이 커서 저장소에 포함하지 않고, `data/sample/`에 대표 CSV별 100줄 샘플만 포함했습니다.

## 주요 스키마

원본 CSV에는 세부 컬럼이 많기 때문에, 전처리와 분석에 사용한 주요 컬럼만 정리했습니다.

컬럼 수는 로컬 `../DataSet`에 있는 대표 CSV 파일의 헤더를 기준으로 확인했습니다.

### 생활인구 데이터

대표 파일 기준 컬럼 수: 32개

- 기준일자
- 시간대
- 행정동 코드
- 총생활인구수
- 연령대별 생활인구

### 지하철 승하차 데이터

대표 파일 기준 컬럼 수: 27개

- 사용일자
- 호선명
- 역명
- 시간대별 승차 인원
- 시간대별 하차 인원

### 지하철 역사 마스터 데이터

분석 사용 컬럼 수: 4개

- 역사명
- 호선
- 위도
- 경도

### 상권 추정매출 데이터

대표 파일 기준 컬럼 수: 55개

- 기준년분기
- 상권 코드
- 상권명
- 서비스 업종 코드
- 서비스 업종명
- 시간대별 매출 금액
- 연령대별 매출 금액

### 상권/행정동 영역 데이터

대표 파일 기준 컬럼 수: 5개

- 행정동 코드
- 행정동명
- 엑스좌표
- 와이좌표
- 영역 면적

### 역-행정동 매핑 데이터

생성 CSV 컬럼 수: 7개

- station_name
- line_name
- latitude
- longitude
- dong_code
- dong_name
- source
